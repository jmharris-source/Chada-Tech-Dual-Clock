# Chada Tech Dual Clock Project

## Project Overview

This project was created as part of a simulated junior developer coding skills test for Chada Tech. The goal of the project was to create a C++ application that displays both a 12-hour clock and a 24-hour clock at the same time.

The program allows the user to adjust the time by adding one hour, one minute, or one second through a menu-driven interface.

## Scenario

Chada Tech works with domestic and international clients. Because international clients may use the ISO 8601 time standard, the company wanted a program that could display both standard 12-hour time and 24-hour time.

This application demonstrates the use of C++ functions, modular programming, user input, conditional logic, and formatted output.

## Features

- Displays a 12-hour clock with AM and PM
- Displays a 24-hour clock
- Allows the user to add one hour
- Allows the user to add one minute
- Allows the user to add one second
- Uses a menu-driven interface
- Ends the program when the user selects the exit option
- Uses functions to keep the code organized and modular
- Includes in-line comments explaining the purpose of important code sections

## How to Run the Program

1. Open Microsoft Visual Studio.
2. Create a new empty C++ project.
3. Add a new C++ source file named `Clock.cpp`.
4. Copy the code into the file.
5. Build and run the program.
6. Use the menu options to adjust the clock.
